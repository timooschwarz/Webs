Installationen, Updates und nützliche Tools.

Installation Updates Tools Programme

Software installieren

So installierst du Programme auf deinem Arbeitsgerät:

  1. Öffne das Unternehmensportal.
  2. Suche nach der gewünschten Software.
  3. Klicke auf Installieren.
  4. Folge den Anweisungen am Bildschirm.

Programme auf dem neuesten Stand halten

So hältst du deine Software aktuell:

  • Nutze die automatischen Updates, wenn verfügbar.
  • Prüfe regelmäßig auf Updates im Unternehmensportal.
  • Starte Programme nach Updates neu.

VPN-Verbindung einrichten

So verbindest du dich mit dem Firmen-VPN:

  1. Öffne die VPN-Einstellungen.
  2. Füge eine neue VPN-Verbindung hinzu.
  3. Serveradresse und Anmeldedaten eingeben.
  4. Mit Verbinden starten.

Nützliche Tools

Programme, die im Arbeitsalltag helfen:

  • 7-Zip – Dateien entpacken
  • Adobe Reader – PDF lesen
  • Teams – Videokonferenzen
  • OneDrive – Cloud-Speicher
