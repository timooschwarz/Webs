Installation Updates Tools Fehler

Software installieren

So installierst du Software aus dem Software Center:

  1. Öffne das Software Center.
  2. Suche nach der gewünschten Software.
  3. Klicke auf Installieren.
  4. Warte, bis die Installation abgeschlossen ist.

Updates installieren

So installierst du Windows-Updates:

  1. Gehe zu Einstellungen > Windows Update.
  2. Klicke auf Nach Updates suchen.
  3. Wähle die gewünschten Updates aus.
  4. Klicke auf Herunterladen und installieren.
